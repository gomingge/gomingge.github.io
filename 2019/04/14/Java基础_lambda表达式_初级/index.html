Lambda表达式

一. 概述

1. 为什么有Lambda表达式

  • 面向对象思想强调的是怎么做 : 导致代码过于冗余
  • 函数式编程 : 强调的是做什么

2. 介绍Lambda表达式的格式

1. 标准格式
  • (参数列表) : 一些参数
  • -> : 一个箭头, 将左边的参数给右边使用
  • {} : 一段代码
2. 可推导可省略
  1. 小括号中的数据类型可以省略
  2. 小括号中如果只有一个参数, 小括号可以省略
  3. 大括号中如果只有一条语句, 大括号和分号可以省略(一起省略)
  4. 大括号中如果只有一条语句,且还是返回语句, 大括号, 分号和return可以省略(一起省略)

3. Lambda表达式使用说明

lambda使用的前提条件

  1. 必须存在一个接口, 这个接口中有且只有一个抽象方法
  2. 接口作为方法的参数
